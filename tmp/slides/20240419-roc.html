Joomla! Template Overrides

zet Joomla! naar je eigen hand

Presentatie van Hans Kuijpers

Wat is een template override?

Een stuk code dat ingeladen wordt in plaats van Joomla! core.

Waarom maak je een override?

  • Omdat je iets wil wijzigen. En je maakt geen core hacks.
  • Om te voorkomen dat je een 3th party extensie moet installeren.
  • Zet de code en uiterlijk naar eigen hand.
  • Behoud code bij een upgrade.

Hoe maak je een override?

  • Via Joomla! Administrator
  • Via CLI of File Manager

  • Pagina /login button breed maken
  • Development toolbar gebruiken om het juiste component en view op te sporen.
  • Template override via System > Site Templates > [template] > Create Override
  • Bestand bewerken via System > Site Templates > [template] > Editor
  • className w-100 toevoegen bij de login button
